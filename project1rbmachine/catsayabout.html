Structure结构

The entire website includes a navigation bar, start page, and result page.

整个网站包括导航栏、开始页和结果页面

Navigation Bar导航栏

The navigation bar remains visible at the top of the website. It includes logo text that can navigate back to the start page, and a link to this documentation page.

导航栏自身能够一直显示在网站顶部。包括了可跳转回到开始页的logo文字，以及链接到此说明页面的链接

Reference / 参考教程

Page Switching页面切换

Page display is controlled through grouping.

通过分组来控制显示状态

Start Page开始页

The start page contains text and a start button. Clicking the start button hides the start page and displays the result page.

开始页包含了文字和开始按钮，开始按钮会让开始页面消失，结果页面出现

Result Page结果页

The result page displays images and text fetched from several APIs, along with an Again button that functions the same as the Start button.

结果页包含了从几个API获取的图片和文字，以及与Start按钮功能相同的Again按钮
